보안 프로그램은 그 수를 줄이고 호환성이 있도록 해야만 한다.
우리 사회는 보안 프로그램의 과잉(생산)으로 인해 하나의 서비스를 이용하기 위해서 너무나도 많은 설치 과정을 거치고 나서 이용이 가능한 상황이다. 단순히 보안 프로그램이 많은 것이 문제가 될 수도 있지만 더 심각한 것은 이렇게 많은 보안 프로그램들이 서로 호환이 되지 않는다는 것이다. 이는 다른 서비스를 이용하기 위해선 그 프로그램에서만 제공하는 보안 프로그램을 새로 설치를 해야만 한다는 것을 의미한다. 이 상황이 반복되다 보면 비슷한 프로그램들끼리 충돌이 발생하게 되며 결국에는 우리 일반 사용자들의 컴퓨터에 문제를 일으킴으로써 편리하게 집에서 일을 해결하고자 하는 '온라인'이 가지고 있는 장점이 완전히 퇴색된 상태이다. 이 상황이 얼마나 심각한 것인지 알 수 있는 것들 중 하나가 바로 '구라 제거기'라는 프로그램의 탄생이다. 구라 제거기란 우리가 여태까지 여러 인터넷 서비스를 이용하면서 불가피하게 설치하게 된 보안 프로그램을 스스로 선별 후 삭제해 주는 프로그램이다. 이 프로그램의 존재 자체가 호환되지 않고 개수만 많은 보안 프로그램이 얼마나 우리가 사용하는 데 있어서 얼마나 큰 방해요소이며 또 원활하고 편하게 일처리를 하기 위한 컴퓨터 사용이 이런 프로그램까지 설치를 해야 한다는 것이 인터넷이 가진 본래의 장점을 잃은 것을 확인 할 수 있다.

Due to the overproduction of security programs, our society needs to go through too many installation processes to use a single service. Simply having a lot of security programs may be a problem, but what's more serious is that so many security programs are not compatible with each other. This means that in order to use other services, you must install a new security program that is only available in that program. If this situation is repeated, similar programs will collide with each other, eventually causing problems with our general users' computers, and the advantages of "online" to conveniently solve things at home have completely faded. One of the things that shows how serious the situation is is the birth of a program called 'Gura Deer.' The Gura Purifier is a program that selects and deletes security programs that we have inevitably installed while using various Internet services. The existence of the program itself is an incompatible and only a large number of security programs, a major obstacle to our use, and the fact that we have to use computers to smoothly and conveniently install such programs has lost its original advantage.
